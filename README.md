# react-native-swapi-custom-plugin

swapi plugin

## Installation

```sh
npm install react-native-swapi-custom-plugin
```

## Usage

```js
import { getmovieList } from "react-native-swapi-custom-plugin";

// ...

getmovieList({}, function (data)
 { console.log("Result of getmovieList ", data);
 });
```
